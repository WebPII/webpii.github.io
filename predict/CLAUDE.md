# Prediction Notes

Benchmark accuracy and latency across detectors that localize visual PII from
ground-truth bounding boxes generated by `ui_reproducer/`.

Baselines include Presidio image redaction, OCR plus LLM classification,
Tesseract/PaddleOCR variants, and YOLO detectors trained on WebPII annotations.

Detectors should not receive privileged access to the ground-truth PII values at
evaluation time.
